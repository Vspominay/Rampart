How I can create account?

  If you want to add a new account to the list, you need to go to the corresponding menu item. After filling in all the required fields, click the Create button

How I can generate a strong password?

  In the menu for creating a new entry there is a corresponding block for generating passwords. You need to select the characteristics for the password and click the generate button. The password will be automatically entered into the password field and copied to the clipboard.

How I can update data about profile?

  Being in the storage menu item, click on the desired account, click on the show password button, and then on the edit button. Change the data and confirm the changes. Clicking on each field automatically copies the data.

How I can update delete account from the list?

  Being in the storage menu item, click on the desired account, click on the show password button, and then on the delete button.

How I can change default icon for account card?

  Being in the storage menu item, click on the desired account, click on the current icon, then select the required one from the list and confirm the actions.

How I can delete user account?

In the upper right corner, click on the username, then in the drop-down menu, click on the corresponding item and confirm your actions
